Help

Using the Stopwatch:

  • Click on the Stopwatch button on the main page to access the stopwatch.
  • The stopwatch starts at 00:00:00 000.
  • Click on the Start button (green) to start the stopwatch.
  • Once started, the Start button will change to Pause (orange). Click on it to pause the stopwatch.
  • If the stopwatch is paused, the Pause button will change to Continue (blue). Click on it to resume the stopwatch.
  • Click on the Clear button (red) to reset the stopwatch to 0.
  • Click on the Back button (gray) to return to the main page.

Using the Countdown Timer:

  • Click on the Countdown Timer button on the main page to access the timer.
  • The timer starts at 00:00:00 000.
  • Enter the desired countdown time in the format hh:mm:ss using the input field or the numeric buttons provided.
  • Click on the Set button (green) to start the countdown timer.
  • Once started, the Set button will change to Pause (orange). Click on it to pause the timer.
  • If the timer is paused, the Pause button will change to Continue (blue). Click on it to resume the timer.
  • Click on the Clear button (red) to reset the timer to 0 and clear the input field.
  • Click on the Back button (gray) to return to the main page.
